# C Programming: A Modern Approach - Exercises

This repository contains my personal solutions to the projects and exercises from the book **"C Programming: A Modern Approach"** by K. N. King.

## Project Goal
The main objective is to master the fundamentals of C, manual memory management, and system-level logic as part of my transition towards **Embedded Systems and IoT**.

## Environment
- **OS:** Ubuntu (Linux)
- **Hardware:** Lenovo ThinkPad T480
- **Compiler:** GCC
- **Editor:** Vim / Terminal


## How to Build
I use a `Makefile` to automate the compilation process. To compile all programs, simply run:
```bash
make
